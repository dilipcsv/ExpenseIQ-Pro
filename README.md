## 🚀 Overview

ExpenseIQ Pro is a modern expense management web application that helps users track daily spending, analyze expenses, and gain smart financial insights through interactive dashboards.

Designed with a premium UI and powerful analytics for students, professionals, and personal finance users.

---

## ✨ Features

✅ Add and manage expenses
✅ Beautiful dashboard UI
✅ Monthly expense tracking
✅ Category-wise analytics
✅ Download expenses as CSV
✅ Smart spending insights
✅ SQLite database integration
✅ Fast and lightweight performance

---

## 📊 Dashboard Includes

* 💰 Total Spending
* 📅 Monthly Expense Summary
* 📈 Daily Spending Trends
* 🥧 Category Breakdown Charts
* 🤖 AI-style Expense Predictions

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/ExpenseIQ-Pro.git
cd ExpenseIQ-Pro
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
